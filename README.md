# 🎓 Capgemini Data Engineering Training

This repository serves as the central documentation for my **Data Engineering Training Program** at Capgemini. It follows a structured approach to mastering the modern data stack, including **Databricks**, **Apache Spark (PySpark)**, and **Cloud Data Warehousing**.

---

## 📂 Repository Structure

The repository is organized by weeks and phases to track the progression of technical skills:

* **[Week0/](Week0/)**: Foundation & Environment Setup
    * `Phase1/`: Data Exploration & Selection
    * `Phase2/`: Joins & Aggregations
    * `Phase3/`: Pipeline Automation
* **Week1/**: Advanced Processing & Case Studies (Olist/BFSI)
    * *Upcoming...*

---

## 🛠️ Technical Stack
- **Languages:** Python (PySpark), SQL
- **Platform:** Databricks
- **Storage:** Delta Lake / DBFS
- **Orchestration:** Databricks Jobs / Pipelines

## 📌 Project Standards
Each folder in this repository follows the Capgemini Review Standard:
1. **README.md**: Documentation of problem statement and approach.
2. **Code**: Clean, commented `.py` or notebook files.
3. **Outputs**: Verified screenshots and result sets in the `/outputs` folder.
